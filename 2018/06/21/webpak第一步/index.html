Webpak第一步

创建基本的webpack4.5项目

1.安装webpack

sudo npm install webpack webpack-cli webpack-dev-server -g

2.创建项目目录

mkdir webpack-dome1

cd webpack-dome1

mkdir config dist src

3.初始化项目

npm init -y
357 ls -la

4.创建默认文件

touch dist/index.html src/index.js

5.打包测试

webpack

webpack –mode=development
